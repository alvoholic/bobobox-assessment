Query hanya mengambil lokasi Type = 'Pod' dan Rating = 5
-> Jadi hanya B001 (L01) dan B002 (L02).

Sekarang kita hitung Previous_Check_Out_Date:

Untuk B001 (Alice, 2024-03-05 -> +3 hari = check-out 2024-03-08)
-> Tidak ada booking sebelumnya -> NULL.

Untuk B002 (Alice, 2024-06-15 -> +2 hari = 2024-06-17)
-> Booking sebelumnya oleh Alice: B001 (check-out 2024-03-08).
Karena < 2024-06-15 -> cocok.
MAX(2024-03-08) = 2024-03-08.

Query ini menampilkan semua booking dengan rating 5 di lokasi bertipe “Pod”, sekaligus menghitung tanggal check-out terakhir sebelum booking tersebut untuk setiap tamu.
- Menunjukkan pola tamu loyal yang sering menginap di pod hotel.
- Previous_Check_Out_Date membantu tahu kapan terakhir kali tamu itu menginap sebelumnya.
